# 🚀 SafeOrbit AI

## 🛰️ Overview
SafeOrbit AI is a simulation-based system designed to detect and avoid collisions between satellites and space debris.

## 💡 Features
- Real-time collision detection
- Dynamic path adjustment
- Stable orbital simulation

## ⚙️ How it Works
- Satellites and debris are modeled as 2D points
- Distance is continuously calculated
- If distance < threshold → collision risk
- Satellite path is adjusted automatically
